These lessons are meant to help research assistants and students in Research Methods courses learn to use R to analyze data in Psychology. Instructors can feel free to adapt them for use in your labs or courses. 

If you have never installed R or R Studio before, check out this installation guide. 
[Getting Started with R and R Studio: Installation Guide](https://julia-m-smith.github.io/Getting Started with R and R Studio.pdf)

If you already have R Studio installed, you can open these .rmd files in R Studio. Students can follow along with the lessons and adapt the code to suit their own needs. 

[Lesson 1: Reading in Data, Setting a Working Directory, Making a Project, and Basic Descriptives](https://julia-m-smith.github.io/R_Lesson1.Rmd)
In this lesson, you will learn how to read data into R and run basic descriptive measures to get a sense of your data.

[Basic plots in ggplot2](https://julia-m-smith.github.io/ggplot_lesson.Rmd)
In this lesson, you will learn how to make various plots using the ggplot2 package.
